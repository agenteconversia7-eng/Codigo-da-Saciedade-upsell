# Upsell — Estudos de Caso do Código da Saciedade

Página de upsell pós-compra premium para o produto **Código da Saciedade**.

## Sobre
Apresentada imediatamente após a compra do ebook principal, essa página oferece o complemento **"Estudos de Caso do Código da Saciedade"** — um ebook interativo em formato Gamma.

## Estrutura
- `index.html` — Página principal (10 seções)
- `style.css` — Identidade visual alinhada com a landing page original
- `main.js` — Animações de scroll e interações
- `hero_image.png` — Imagem hero da seção de parabéns
- `estudos_caso_cover.png` — Capa do ebook upsell

## Design
- Paleta: verde floresta + creme + dourado
- Tipografia: Cormorant Garamond + Jost
- Tom: acolhedor, premium, feminino, transformador
- Totalmente responsivo para mobile
